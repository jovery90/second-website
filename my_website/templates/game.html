About Stori:

Written in Python 3.0

Libraries and Modules: Pygame, Tiled, PyTweening, and Itertools

Frozen as .exe file in PyInstaller


After initial study of the programming language, Python, Jonathan took interested in developing a major project. He initially started with

smaller projects that used the skills he was developing in Python and accessed them in the real world. Jonathan thought that a stunning

way to highlight those skills would be via a game. Python programming skills he wanted to highlight with this endeavor included

extensively using OOP (Object Oriented Programming- used to create objects in-game), lists and dictionaries (eg. used for "level-ups" of

the character, to switch weapons , sounds, and functionality), while loops (the game loop itself), Boolean logic (collisions), as well as overall

programming skills and dedication. Jonathan thoroughly enjoyed the beta version creation of Stori and is excited to use Python for more

real-world applications.
